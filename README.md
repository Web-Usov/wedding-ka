# Приглашение на свадьбу — Алексей & Екатерина

Одностраничный сайт-приглашение на свадьбу. Mobile-first, адаптивный.

## Демо

`http://192.168.20.105` (локальная сеть)

## Структура сайта

| Секция | Описание |
|--------|----------|
| Hero | Обложка с именами и датой |
| Приглашение | Текст + таймер обратного отсчёта |
| Дресс-код | Цветовая палитра + 10 образов |
| Детали | Цветы, подарки, дети, сюрпризы, ведущий |
| Тайминг | Расписание дня |
| Локация | Адрес + карта |
| RSVP | Форма подтверждения участия |
| Фото | QR-код на загрузку фото |

## Технологии

- React 19 + Vite
- Tailwind CSS v4
- Framer Motion
- Docker + Nginx

## Запуск

```bash
docker compose up -d
```

## Обновление

```bash
cd frontend && npm run build && cd .. && docker compose restart nginx
```
