# Спецификация: Цифровое приглашение на свадьбу

## 1. Общая информация

| Параметр | Значение |
|----------|----------|
| **Тип проекта** | Одностраничный сайт / цифровое приглашение |
| **Формат** | Вертикальный, mobile-first (соотношение 9:16 как у Stories), адаптивный под десктоп |
| **Язык** | Русский |
| **Стиль** | Минимализм, элегантность, «quiet luxury», природная эстетика |
| **Основной формат подачи** | Scrollable single-page с полноэкранными секциями |
| **Контент на этапе MVP** | Заглушки из референсов (см. раздел 4 и 7). Финальное заполнение — отдельный этап |

---

## 2. Структура страницы

Страница состоит из 7 последовательных секций (экранов):

1. **Hero / Обложка** — титульный экран с именами и датой
2. **Обращение + таймер** — пригласительный текст и обратный отсчёт
3. **Дресс-код** — описание стиля, цветовая палитра, примеры
4. **Детали мероприятия** — цветы, подарки, дети, сюрпризы
5. **Тайминг** — расписание дня
6. **Локация** — адрес и интерактивная карта
7. **RSVP** — форма подтверждения участия

## 3. Дизайн-система

### 3.1 Типографика

| Элемент | Шрифт | Начертание | Примечания |
|---------|-------|------------|------------|
| Заголовки секций | Каллиграфический скрипт | Regular | Завитки, имитация пера (например, Great Vibes, Allura, Playfair Display Italic) |
| Имена (Hero) | Каллиграфический скрипт | Regular | Вертикальная композиция с амперсандом |
| Рубрики / подзаголовки | Геометрический sans-serif | Light / Regular | Капс (ALL CAPS), увеличенный трекинг |
| Основной текст | Геометрический sans-serif | Light / Regular | Комфортная межстрочка (~1.5–1.6) |
| Декоративные цифры (тайминг) | Классическая антиква | — | Крупные, полупрозрачные, на фоне |

**Рекомендуемые пары:**
- Скрипт: `Great Vibes` или `Allura`
- Основной: `Montserrat` (Light/Regular) или `Inter`

### 3.2 Цветовая палитра

| Назначение | Значение |
|------------|----------|
| Фон (текстовые секции) | `#FFFFFF` (чистый белый), `#F5F5F5` (светло-серый) |
| Фон (Hero) | Природное фото с blur/bokeh и тёмной виньеткой |
| Фон (тайминг) | Тёмное фото с затемнением (`rgba(0,0,0,0.5)`) |
| Текст на светлом фоне | `#1A1A1A` (чёрный), `#333333` (тёмно-серый) |
| Текст на тёмном фоне | `#FFFFFF` (белый) |
| Плейсхолдеры | `#888888` (серый средний) |
| Цвета дресс-кода | Таупе, белый, оливковый/хаки, чёрный |
| Акценты | Приглушённые натуральные тона, без ярких красок |

### 3.3 UI-элементы

- **Кнопки**: закруглённые прямоугольники (`border-radius: 8–12px`), белые с тёмным текстом или тёмные с белым текстом
- **Поля ввода**: белые, закруглённые, тонкая рамка или без рамки, серые плейсхолдеры
- **Чекбоксы**: квадратные с тонкой рамкой (`1px`), без заливки, белые на тёмном фоне
- **Иконки**: тонкий линейный арт (outline), stroke `1–1.5px`, без заливки
- **Разделитель**: тонкая вертикальная серая линия (декоративный акцент)

## 4. Описание секций

> **MVP-нотация:** Весь текстовый и визуальный контент в секциях ниже — это заглушки, взятые из референсов. Они используются для верстки и демонстрации структуры. Финальные тексты, имена, даты, адреса, фото и контакты заменяются на отдельном этапе.

### 4.1 Hero / Обложка

**Фон:** Полноэкранное фото туманного леса/аллеи. Сильное размытие (bokeh), мягкая дымка, тёмная виньетка по краям. Формат `cover`.

**Контент (центрирован по вертикали и горизонтали):**
- Верх: `ПРИГЛАШЕНИЕ НА СВАДЬБУ` — sans-serif, капс, трекинг, белый
- Центр: `Имя & Имя` — каллиграфический скрипт, белый. Вертикальная композиция: первое имя выше, амперсанд по центру, второе имя ниже
- Низ: `15 июня 2024` — sans-serif, белый

**Анимация:** Плавное fade-in текста при загрузке. Возможен лёгкий параллакс фона при скролле.

---

### 4.2 Обращение + таймер обратного отсчёта

**Фон:** Чистый белый (`#FFFFFF`) или светло-серый (`#F5F5F5`).

**Контент (центрирован):**
- Заголовок: `Дорогие родные и близкие!` — sans-serif, light, крупный кегль
- Текст: 2 абзаца пригласительного текста — sans-serif, regular, средний кегль
- Подзаголовок: `Наша свадьба состоится через:` — sans-serif, regular
- **Таймер:** 4 колонки в ряд:
  - `00` Дней
  - `00` Часов
  - `00` Минут
  - `00` Секунд
  - Цифры крупные (bold/light), подписи мелкие
- Акцент внизу: `Дресс-код` — каллиграфический скрипт, единственный декоративный элемент

**Функционал:** Таймер автоматически обновляется каждую секунду. **Целевая дата (MVP):** `15 июня 2024, 15:00`.

---

### 4.3 Дресс-код

**Фон:** Белый / светло-серый.

**Контент (центрирован):**
- Заголовок: `Дресс-код` — каллиграфический скрипт
- Текст: `Для нас главное — ваше присутствие! Но мы будем рады, если в своих нарядах вы поддержите цветовую гамму и стиль нашей свадьбы.` — sans-serif, центрирован
- **Цветовая палитра:** 4 круга одинакового размера в ряд:
  1. Таупе (приглушённый серый)
  2. Белый (с тонкой обводкой)
  3. Оливковый / хаки
  4. Чёрный
- **Галерея:** 2 фото в форме скруглённых прямоугольников (сильное закругление, почти капсулы), перекрывающие друг друга с небольшим смещением:
  - Левое: мужской образ
  - Правое: женский образ (лежит поверх левого)

### 4.4 Детали мероприятия

**Фон:** Светло-серый (`#F5F5F5` или `#E8E8E8`), однотонный, эффект матовой бумаги.

**Контент (центрированная колонка, выровненная по левому краю внутри):**
- Заголовок: `Детали` — каллиграфический скрипт

**Блоки (с иконками слева):**

1. **Цветы** (иконка: роза/веточка, линейный арт)
   - Заголовок: `ЦВЕТЫ` — sans-serif, капс
   - Текст: `Наш праздник будет окружён изобилием цветов, поэтому приятным комплиментом для нас вместо букета будет бутылка вашего любимого вина, которую мы откроем на ближайшем совместном празднике.`

2. **Что дарить?** (иконка: подарочная коробка с бантом)
   - Заголовок: `ЧТО ДАРИТЬ?` — sans-serif, капс
   - Текст: `В качестве подарка будем рады вкладу в бюджет нашей семьи. Он точно поможет воплотить нашу мечту в реальность!`

3. **Маленькие гости** (иконка: взрослая и детская руки)
   - Заголовок: `МАЛЕНЬКИЕ ГОСТИ` — sans-serif, капс
   - Текст: `Дорогие гости, мы будем благодарны, если вы посвятите нашему дню всё своё внимание и энергию. Поэтому будем признательны, если наш праздник пройдёт в кругу взрослых.`

4. **Сюрпризы** (иконка: фейерверк/бенгальский огонь)
   - Заголовок: `СЮРПРИЗЫ` — sans-serif, капс
   - Текст: `Возьмите с собой отличное настроение, приветствуются ваши поздравления, активное участие в интерактивах, творческие выступления!`
   - Контакт: `Анастасия +7 922 254 66 50`

**Композиция:** Щедрые воздушные интервалы между блоками. Иконки слегка выступают в левое поле.

### 4.5 Тайминг

**Фон:** Тёмное фото интерьера (хрустальная люстра) с сильным затемнением (`rgba(0,0,0,0.6)`). Виньетка. Формат `cover`.

**Контент:**
- Заголовок: `Тайминг` — каллиграфический скрипт, белый

**События (вертикальный список с волнообразным смещением):**

| Время | Событие |
|-------|---------|
| `15:00` | СБОР ГОСТЕЙ |
| `15:30` | ТОРЖЕСТВЕННАЯ ЦЕРЕМОНИЯ |
| `16:00` | ФУРШЕТ |
| `17:00` | ПРАЗДНИЧНЫЙ УЖИН |

**Визуальная техника:** Каждое событие оформлено с эффектом «двойных цифр»:
- Крупные полупрозрачные цифры (`rgba(255,255,255,0.15)`) как фоновый декор
- Поверх них — чёткие белые цифры меньшего размера
- Под ними — подпись капсом, белая, sans-serif

**Композиция:** Стаггерed (волнообразное смещение по горизонтали): первое событие правее центра, второе — левее, и т.д.

### 4.6 Локация

**Фон:** Светло-серый градиент, гладкий, без текстур.

**Контент (центрирован в верхней части):**
- Заголовок: `Локация` — каллиграфический скрипт
- Подзаголовок: `Ресторан «Название»` — sans-serif, regular
- Адрес: `Улица, номер дома` — sans-serif, regular

**Карта:**
- Нижняя часть экрана (40–45% высоты) — встроенная интерактивная карта
- Сервис: Яндекс.Карты или Google Maps
- Маркер места проведения
- Кнопка «Построить маршрут» / «Как добраться»

### 4.7 RSVP — Форма подтверждения участия

**Фон:** Фото с природой (зелёная листва, белые цветы, элементы декора) или тёмно-зелёный фон. Формат `cover`.

**Контент (центрированная колонка):**
- Заголовок: `Мы ждём именно вас!` — sans-serif, white
- Дата/время: `15 июня | Суббота | 15:00` — sans-serif, white

**Поля формы (белые блоки с закруглёнными углами):**

| Поле | Тип | Плейсхолдер | Обязательность |
|------|-----|-------------|----------------|
| Имя Фамилия | text | `Имя Фамилия` | Обязательно |
| Телефон | tel | `Ваш телефон` | Обязательно |
| Аллергия | text | `Есть ли у вас аллергия? Если да, укажите на что` | Опционально |
| +1 (пара) | text | `Приду с парой (имя, фамилия)` | Опционально |

**Вопросы:**
- `Необходимо ли вегетарианское меню?` — dropdown: `Нет` / `Да`
- `Сможете ли присутствовать?` — dropdown: `С радостью приду!` / `К сожалению, не смогу`

**Чекбоксы:**
- Заголовок: `Буду пить`
- Опции (множественный выбор):
  - [ ] Мартини
  - [ ] Вино красное
  - [ ] Вино белое
  - [ ] Виски
  - [ ] Водка
  - [ ] Не буду пить алкоголь

**Кнопка:** `Отправить` — белая/тёмная, закруглённая, полная ширина формы

**Функционал:**
- Валидация обязательных полей
- Сообщение об успешной отправке (например, `Спасибо! Мы получили ваш ответ ❤️`)

## 5. Анимации и эффекты

| Эффект | Применение | Технология |
|--------|------------|------------|
| **Fade-in + slide-up** | Появление текстовых блоков при скролле | Framer Motion / GSAP ScrollTrigger |
| **Таймер обратного отсчёта** | Автообновление каждую секунду | Vanilla JS / React state |
| **Параллакс фона** | Hero и тайминг — фон движется медленнее контента | CSS `background-attachment: fixed` или GSAP |
| **Плавное появление Hero** | Загрузка страницы — staggered анимация текста | Framer Motion |
| **Hover-эффекты** | Кнопки, поля формы — лёгкое затемнение / подсветка | CSS transitions |
| **Фокус на поле** | Подсветка рамки при вводе | CSS `:focus` |

### 6.2 Форма и данные

**Выбранное решение:** Google Sheets + Google Apps Script

| Компонент | Описание |
|-----------|----------|
| **Google Sheets** | Хранение всех RSVP-ответов |
| **Google Apps Script** | Web-приложение (doPost), принимает JSON из формы и записывает в таблицу |
| **CORS** | Разрешённый origin — либо `*`, либо домен свадебного сайта |

**Эндпоинт:** `https://script.google.com/macros/s/SCRIPT_ID/exec`
**Метод:** `POST`, `Content-Type: application/json`

**Поля, отправляемые в таблицу:**
| Колонка | Поле |
|---------|------|
| A | Имя Фамилия |
| B | Телефон |
| C | Аллергия |
| D | +1 (пара) |
| E | Вегетарианское меню |
| F | Присутствие (да/нет) |
| G | Алкоголь (список через запятую) |
| H | Дата и время отправки |

### 6.3 Карта

**Выбранное решение:** Яндекс.Карты

| Компонент | Описание |
|-----------|----------|
| **Yandex Maps Static API** | Статичное изображение карты с маркером |
| **Ссылка** | При клике — открытие Яндекс.Карт с маршрутом (`https://yandex.ru/maps/?rtext=~{lat},{lon}`) |

**Альтернатива:** Yandex Maps JS API (Widget) — если требуется интерактивная карта прямо на странице.

## 8. Адаптивность

### Mobile (основной формат)
- Вертикальная ориентация, полноэкранные секции
- Контент центрирован, щедрые отступы по бокам (`16–24px`)
- Текст комфортного кегля (не мельче `14px`)
- Карта — интерактивная, занимает нижнюю часть экрана
- Форма — поля на всю ширину с комфортными отступами

### Tablet
- Центрирование контента с ограничением максимальной ширины (`max-width: 600px`)
- Фон по бокам — продолжение основного фона или градиент

### Desktop
- Максимальная ширина контента: `480–600px`
- Фон по бокам: приглушённый градиент, текстура, или blur-версия основного фона
- Центрирование всей композиции
